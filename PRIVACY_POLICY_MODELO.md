# Política de Privacidade - Coringa Plus

**Última atualização:** 09 de Janeiro de 2026

A privacidade dos visitantes e usuários é de extrema importância para nós. Esta Política de Privacidade descreve os tipos de informações pessoais que são coletadas e recebidas pelo aplicativo **Coringa Plus** e como elas são utilizadas.

## 1. Informações que Coletamos

O aplicativo **Coringa Plus** foi desenvolvido para fins corporativos de registro de ponto e gestão de jornada. Para cumprir sua função, o aplicativo necessita coletar os seguintes dados:

### 1.1. Câmera e Imagens
O aplicativo solicita permissão de acesso à câmera do dispositivo.
*   **Finalidade:** A câmera é utilizada exclusivamente para capturar uma fotografia (selfie) do colaborador no momento do registro do ponto.
*   **Uso:** Esta imagem serve para autenticação visual e validação da identidade do usuário perante o empregador.

### 1.2. Localização (GPS)
O aplicativo solicita permissão de acesso à localização (Geolocalização).
*   **Finalidade:** Registrar as coordenadas geográficas (latitude e longitude) exatas de onde o registro de ponto está sendo realizado.
*   **Uso:** Garantir que o colaborador está no local de trabalho designado ou em rota autorizada.

### 1.3. Dados do Dispositivo
Podemos coletar informações técnicas sobre o dispositivo, como modelo, sistema operacional e identificadores únicos, para garantir a segurança e evitar fraudes no registro.

## 2. Como Usamos as Informações
As informações coletadas são utilizadas exclusivamente para:
*   Gerar o comprovante de registro de ponto conforme exigências legais (Portaria 671/2021 do MTP ou legislações vigentes).
*   Enviar os dados para o sistema de gestão de RH da empresa empregadora.
*   Prevenir fraudes na marcação de ponto.

**Não** utilizamos seus dados para fins publicitários, marketing ou venda para terceiros.

## 3. Compartilhamento de Dados
Os dados coletados (foto, localização e horário) são compartilhados com:
*   **A empresa empregadora:** Que é a controladora dos dados para fins trabalhistas.
*   **Servidores de Nuvem:** Onde os dados são armazenados de forma segura e criptografada para acesso do RH.

## 4. Segurança
Adotamos medidas de segurança adequadas para proteção contra acesso não autorizado, alteração, divulgação ou destruição dos dados. A comunicação entre o aplicativo e os servidores é realizada via protocolo seguro (HTTPS).

## 5. Contato
Se você tiver dúvidas sobre esta Política de Privacidade, entre em contato com o suporte técnico da sua empresa ou através do desenvolvedor do aplicativo.
